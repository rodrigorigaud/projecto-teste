Neste tutorial, iremos criar uma aplicação Web de Gestão de utilizadores CRUD (Create Read Update Delete) simples, utilizando Jsp, Servlet e MySQL.

Para este tutorial, precisamos das seguintes ferramentas: (A versão mais antiga ou mais recente também deve funcionar). Além disso, o conhecimento Java básico é assumido.

1. Eclipse IDE para desenvolvedores de Java EE (Indigo - versão 3.7)

2. Apache Tomcat ver 7.0

3. MySQL Community Server e MySQL Workbench (Ferramenta GUI)

4. MySQL Connector para Java

5. jstl.jar e standard.jar. Você pode obter esses frascos de seu Tomcat. Verifique neste diretório: (seu diretório tomcat) -> apache-tomcat-7.0.26-windows-x86-> apache-tomcat-7.0.26-> webapps-> examples-> WEB-INF-> lib

Vou lhe dizer onde você deve colocar esses frascos mais tarde.

6. jQuery para a capacidade de javascript. Neste caso, nós apenas usá-lo para o componente datepicker
